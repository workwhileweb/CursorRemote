## You're all set!

CursorRemote is configured and running. Open the web client on your phone to start controlling Cursor remotely.

### Quick reference
- **Status bar** — shows connection state at a glance (click to toggle server)
- **Activity bar** — the CursorRemote panel shows windows, model, and pending approvals
- **Command Palette** — all commands start with `CursorRemote:`
- **Output panel** — select *CursorRemote* for server logs

Happy coding from your couch!
